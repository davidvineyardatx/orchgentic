# Post-v1 Roadmap

This roadmap is intentionally separate from the v1.0.0 release scope.

## v1.1 candidates

- Improve plugin ergonomics.
- Expand example library.
- Improve CLI help text.
- Add more troubleshooting recipes.
- Improve provider setup docs.
- Add optional notification hooks if small and low-risk.

## v1.2 candidates

- Workflow runtime improvements where proven necessary.
- Better trace browsing.
- More robust observability commands.
- Additional production deployment guidance.

## Hosted product candidates

- Dashboard.
- No-code workflow builder.
- Hosted agent execution.
- Multi-user workspace.
- Marketplace.
- Enterprise auth.
- Billing.

## Scope rule

Future work should remain milestone-driven. Avoid pulling hosted/no-code scope into the open-source developer framework unless it directly supports the framework’s adoption path.
