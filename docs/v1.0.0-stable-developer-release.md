# Orchgentic v1.0.0 Stable Developer Release

## Release purpose

v1.0.0 establishes Orchgentic as a stable developer framework for building agentic systems with declarative configuration, runtime validation, tool execution, memory, knowledge/RAG, providers, teams, and workflows.

## Core promise

Developers should be able to:

1. install Orchgentic
2. initialize a project
3. create an agent
4. configure providers, memory, knowledge, and tools
5. run agents and teams
6. validate workflows
7. use doctor-style checks to catch common configuration issues
8. rely on stable docs and examples

## Stable baseline

The stable baseline includes:

- agent runtime
- team runtime
- tool runtime
- provider abstraction
- memory layer
- knowledge/RAG layer
- local deterministic routing
- workflow contract and examples
- stabilization checks
- clean install docs
